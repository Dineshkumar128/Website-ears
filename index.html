Project overview

What we built

EARS enables university staff to log, manage, and report on engagement activities — conferences, workshops, community events, and more.

Zero-dependency frontend
Vanilla HTML, CSS and JS — no React, no bundler. Pure browser-native code that loads in milliseconds.
Progressive Web App
Installable on any device via Web App Manifest + Service Worker. Fully offline after first visit.
Secure authentication
5-rule password policy, XSS sanitisation, session management. Passed external security review.
Google Sheets sync (ready)
Full sync + merge logic written. Activates when admin configures a Google Apps Script URL.
Offline-first cache
Cache-then-Network strategy. All assets pre-cached. Users always get a response, even without signal.
JSON export / import
One-click backup to timestamped JSON. Import merges intelligently with conflict resolution.
